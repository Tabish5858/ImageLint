# Changelog

## 0.1.0 - 2026-03-07

- Initial ImageLint MVP scaffold.
- Added workspace scanner and analyzer for image issues.
- Added optimizer support for raster (`sharp`), SVG (`svgo`), and GIF (`gifsicle`).
- Added inline diagnostics and quick fixes.
- Added audit report WebView with `Fix All` action.
- Added status bar summary and file watcher integration.
- Added CI workflow, test scaffold, and project documentation.
