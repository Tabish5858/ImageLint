# Support

## How to Get Help

For usage questions, troubleshooting, and setup help:

- Open a GitHub issue:
  https://github.com/Tabish5858/ImageLint/issues
- Include relevant environment details and reproduction steps.
- Include logs, screenshots, and a minimal reproduction where possible.

## Before Opening an Issue

- Read README.md and CONTRIBUTING.md.
- Check existing open and closed issues.
- Confirm you are on a supported version.
- Confirm whether the problem reproduces after running `npm ci` and `npm run build`.

## Security Issues

Do not open public issues for vulnerabilities.

Follow SECURITY.md for private disclosure instructions.

## Response Expectations

This is a community-maintained project. Response times may vary, but issues with
clear reproduction details are prioritized.
